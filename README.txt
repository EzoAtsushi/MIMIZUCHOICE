MIMIZU CHOICE - PWAファイル一式

同じフォルダに必ず置くもの:
- index.html
- sw.js
- manifest.webmanifest
- icon-192.png
- icon-512.png
- apple-touch-icon.png
- favicon-32.png
- planck.min.js  ← これはあなたが既に保存したローカルファイルを同階層に置いてください

注意:
- PWAとしてインストールできるのは https または localhost のみです。
- GitHub Pages に置くのが一番簡単です。
- iOSは「共有」→「ホーム画面に追加」でインストールします。

パスワード機能:
- BEST(最速)が出ると PASSWORD 欄にドラクエ風の文字列が表示されます。
- COPYでコピーしてメモ等に保存できます。
- 復元したいときは PASSWORD 欄の入力に貼り付けて LOAD。
  次にSTARTしたとき、その個体をベースにレースが始まります。
